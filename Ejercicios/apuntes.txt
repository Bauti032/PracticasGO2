¿Porque se hace un append?
Cuando se declara el slice no tiene elementos, peor trabaja con un vect interno, cada vez que se hace un append se agrega elementos en los vectores internos.
s = append(s,1) -> verifica el vector interno
                    -> el vector es retornado como resultado de la funcion 


s = append(s,3) -> Se crea otro vector interno con el doble del espacio que el anterior y agrega el elemento nuevo, copiando el anterior

s = append(s, 5) -> Agarra el vector anterior, lo duplica y en vez de tener 2 espacios ahora pasa a tener 4.
